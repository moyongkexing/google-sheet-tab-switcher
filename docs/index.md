# Google Sheet Tab Switcher

Google Sheet Tab Switcher is a Chrome extension for quickly searching and switching tabs inside Google Sheets.

## Purpose

Quickly search and switch visible sheet tabs inside Google Sheets.

## Privacy Summary

The extension reads visible Google Sheets tab labels locally in the browser. It does not collect, store, sell, or transmit user data.

See the [privacy policy]({{ '/privacy/' | relative_url }}) for details.

# Privacy Policy

Last updated: 2026-05-01

Google Sheet Tab Switcher is a Chrome extension for quickly searching and switching tabs inside Google Sheets.

## Data Access

The extension runs on Google Sheets pages that match:

```txt
https://docs.google.com/spreadsheets/*
```

On those pages, it reads visible sheet tab labels from the Google Sheets page interface so it can show a local tab switcher. It also dispatches a local click-like interaction to switch to the selected sheet tab.

The extension does not intentionally read spreadsheet cell contents.

## Data Collection

The extension does not collect, store, sell, or transmit user data.

The extension does not use analytics, ads, tracking, external APIs, cloud sync, or remote logging.

## Data Sharing

The extension does not share user data with third parties.

## Remote Code

The extension does not execute remotely hosted code.

## Permissions

The extension uses host access for Google Sheets so the content script can run only on spreadsheet pages. This access is required for the extension's single purpose: searching and switching sheet tabs.

## Changes

If the extension ever adds analytics, sync, account features, external services, or broader page access, this policy and the Chrome Web Store privacy disclosures must be updated before release.

# Support

Use GitHub issues for support requests, bug reports, and feature proposals.

Before opening an issue:

- Confirm the problem happens on a Google Sheets URL matching `https://docs.google.com/spreadsheets/*`.
- Run the latest available version of the extension.
- Check whether the issue reproduces after reloading the Google Sheets page.

For security-sensitive reports, follow `SECURITY.md` instead of posting details publicly.
